# music-player

This is a music player project written on ReactJS to learn more about hooks, state management and local storage.

# Features

- **Music Playback:** Browse through and play songs with simple playback controls.
- **Playlist Management:** Create, edit and organise playlists for a personalised listening experience and quick access to favorite tracks.

# Installation:

1. 
```
git clone https://github.com/antonVosc/music-player.git
```

2. 
```
cd music-player/
```

3. 
```
npm install
```

# Usage:

```
npm run dev
```
